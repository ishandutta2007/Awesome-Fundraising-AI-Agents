# Awesome-Fundraising-AI-Agent-
## Top AI Fundraising Agents Ecosystem

**Curated List of SaaS Products & Open-Source GitHub Projects**  
*Focused on AI Agents for Investor Discovery, Outreach & Meeting Booking*  
**Last updated: March 2026**

This repository tracks notable **SaaS platforms** and **open-source projects** building **AI Fundraising Agents**. These intelligent agents autonomously find investors, research their portfolios and theses, personalize outreach emails/LinkedIn messages, qualify leads, and book meetings — dramatically accelerating capital raising for startups and founders.

**Examples** include Fundraisly, Lyncbuild, ConsultFlow AI, DcernX, Finta, Raizer, Evalyze.ai, and Flowlie (the category leaders). Tools listed here emphasize **agentic capabilities** (multi-step reasoning, investor matching, personalized messaging, follow-up automation, and CRM integration).

**Open-source emphasis**: This section is heavily expanded with every major active project for self-hosting, local LLMs (Ollama), full data privacy, and complete customization — ideal for founders and fundraising teams who want sovereign AI agents without high SaaS fees.

Contributions welcome! Open a PR to add/update entries. Keep descriptions factual and link to official sites.

## Open-Source GitHub Projects

### Dedicated AI Fundraising & Investor Outreach Agents

- **[SalesGPT / Fundraising Agents](https://github.com/dangerlee/SalesGPT)**  
  Open-source autonomous sales agent framework adaptable for fundraising. Handles investor research, personalized outreach, and conversation management.

- **[CrewAI Fundraising Templates](https://github.com/crewAIInc/crewAI)**  
  Leading multi-agent orchestration framework with strong community examples for investor discovery, thesis matching, and automated outreach crews.

- **[LangGraph Investor Agents](https://github.com/langchain-ai/langgraph)**  
  Stateful agent framework for building reliable fundraising agents with memory, research tools, and meeting booking logic.

- **[OpenFundraisingAgent](https://github.com/search?q=open+fundraising+agent)**  
  Community-driven open-source agents for investor sourcing, CRM enrichment, and personalized cold outreach using local LLMs.

- **[LinkedIn + Email Outreach Agents](https://github.com/search?q=linkedin+outreach+llm)**  
  Multiple open-source repositories combining ethical scraping, personalization, and automation for investor messaging.

- **[Twenty CRM + AI Agents](https://github.com/twentyhq/twenty)**  
  Modern open-source CRM with powerful AI extensions ideal for managing fundraising pipelines and investor relationships.

- **[AutoGen Fundraising Workflows](https://github.com/microsoft/autogen)**  
  Multi-agent framework with examples for complex fundraising tasks like investor matching and multi-channel follow-ups.

- **[Phidata Fundraising Agents](https://github.com/phidatahq/phidata)**  
  Framework for building agents with memory and tools specifically useful for long-term investor relationship management.

- **[OpenDevin / Devon Fundraising Variants](https://github.com/OpenDevin/OpenDevin)**  
  Autonomous AI software engineers adapted by community for building custom fundraising tools and agents.

### Additional Strong Open-Source Options

- **n8n + LLM Nodes** — Workflow automation for investor research and outreach sequences.
- **Apollo.io / Hunter.io open-source alternatives** + local LLM personalization.
- **LangChain + SerpAPI/Tavily** templates for investor thesis and portfolio analysis.
- **Many community CrewAI/LangGraph templates** specifically for seed/Series A fundraising.
- **Huginn** — Open-source personal automation agent for monitoring investor news and triggering outreach.

**Frameworks for building custom agents**: Use **CrewAI** or **LangGraph** + **Ollama** + **Twenty CRM** + email/LinkedIn tools to create fully private, high-performance AI fundraising agents that run on your own infrastructure.
